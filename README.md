# About this project

Code to work along the Data Structures and Algorithms in JavaScript book from Federico Kereki

# Getting started

Simply run npm install to install all dependencies:

```shell
npm install
```

# Useful commands

## Run tests

```shell
npm run test
```

## Run tests in watch mode

```shell
npm run test:watch
```
